4. Verstoß gegen verschiedene PEP8 Style Guide

   - Keine Docstrings
5. Es gibt verschieden Python Hilfspaketen 
   - pylint -> pip install pylint
   - black -> pip install black (Code Formatierung)

   - isort - pip install isort (Importe zu sortieren in der richtige Reihenfolge)
     Ausführen. isort .   Punkt zeig auf Root Verzeichniss
   - mypy -> pip install mypy (hilft z.B bei Type Annotation) 
6. Docstrings: VSC -> Extensions -> docstring -> Python Docstrins Generator 
   Nach der Installation von docstins -> Einstellung in "Rädchen" -> Erweiterungseinstellung -> Wähle DocString Format
